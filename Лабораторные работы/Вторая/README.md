# Лабораторная работа №2 — Прототип многотабличной базы данных в памяти (In-Memory DB)

### 1. Структура проекта
* **Лабораторные работы / Вторая / main.py** — основной скрипт программы. Содержит логику инициализации баз данных двух таблиц, CRUD-операции и консольное меню управления.
* **Лабораторные работы / Вторая / README.md** — текущий файл с описанием проекта и инструкцией.

### 2. Реализованная функциональность
Программа реализует полноценную базу данных в оперативной памяти с поддержкой связей между сущностями.
* **Две таблицы**: Компьютерные игры (`games_db`) и Студии-разработчики (`studios_db`). Таблица игр связана с таблицей студий через поле `studio_id`.
* **Полный цикл CRUD-операций** для обеих таблиц:
  * **C (Create)**: Добавление игр и студий с автоматической генерацией ID.
  * **R (Read)**: Просмотр списков, а также фильтрация игр по жанру или году выпуска.
  * **U (Update)**: Интерактивное обновление любых полей существующих записей по их ID.
  * **D (Delete)**: Удаление записей из базы данных по ID.
* **Валидация данных**: Встроена защита от некорректного ввода типов (например, букв в числовые поля).

### 3. Инструкция по запуску
1. Откройте терминал в папке проекта.
2. Запустите скрипт командой:
   ```bash
   python main.py
   ```
3. Выберите операции с помощью числового меню (пункты 1-9).
