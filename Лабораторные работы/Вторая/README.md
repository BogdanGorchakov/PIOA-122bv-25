# Лабораторная работа №2 — Прототип базы данных в памяти (In-Memory DB)

### 1. Структура проекта
* **Лабораторные работы / Вторая / main.py** — основной скрипт программы. Содержит логику инициализации базы данных, функции добавления/фильтрации записей и консольный интерфейс.
* **Лабораторные работы / Вторая / README.md** — текущий файл с описанием проекта и инструкцией.

### 2. Реализованная функциональность
Программа имитирует работу реляционной базы данных в оперативной памяти ноутбука (без сохранения в файлы).
* Предметная область: **База данных компьютерных игр**.
* Настроена структура таблицы: уникальный ID (генерируется сам), Название, Жанр, Год выпуска.
* Поддерживается операция **Добавления записей** с автоматическим инкрементом ID.
* Поддерживается операция **Чтения и фильтрации** (можно вывести все игры сразу, отфильтровать строго по жанру или найти игры конкретного года выпуска).
* Встроена **обработка ошибок**: программа не вылетает при вводе букв вместо года выпуска, а также проверяет пустые поля при создании игры.

### 3. Инструкция по запуску
1. Установите интерпретатор Python (версии 3.x).
2. Откройте терминал в папке проекта и выполните команду:
   ```bash
   python main.py
   ```
3. Для управления используйте текстовое меню в консоли (пункты 1-3).
